Buster OS - A fast and aesthetic distro

Category : FUN
Completion : March 2019
Role : Solo Developer

This project was started by my long term desire to make my own linux distro. However this was no possible with my skills as a 12 year old. Now I am able to finally create my own distro. After some research I decided to use the book Linux from Scratch to start my own distro off. This was actually much easier than expected I but when I finished I relized without a package manager my distro would never be more usefull than a experiment. I decided to pivot to a fedora based distrubution. Now my distro is almost complete.
